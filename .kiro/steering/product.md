# Product Overview

A mobile-first convenience store e-commerce web application (Thai market focus).

## Core Features

- Product browsing with categories and search
- Flash sales and promotions
- Shopping cart management
- User authentication and profiles
- Order placement and history
- Coupon system
- Delivery address management

## Target Users

Thai consumers shopping for convenience store items (food, drinks, personal care, household goods).

## Key UX Patterns

- Mobile-optimized layout (max-width 480px)
- Bottom navigation for primary actions
- Fixed header with search and cart access
- Thai language primary, English secondary for product names
