# กฎการใช้ภาษา

## ภาษาหลัก

- ใช้ภาษาไทยเป็นหลักในทุกส่วนที่เป็นไปได้
- คำอธิบาย, comments, และข้อความแสดงผลต้องเป็นภาษาไทย

## ข้อยกเว้น

- ชื่อตัวแปร, ฟังก์ชัน, และ class ใช้ภาษาอังกฤษตามมาตรฐาน
- คำศัพท์เทคนิคที่ไม่มีคำแปลที่เหมาะสมสามารถใช้ภาษาอังกฤษได้
- ชื่อไฟล์และโฟลเดอร์ใช้ภาษาอังกฤษ

## ตัวอย่าง Comments

```javascript
// ดึงข้อมูลสินค้าจาก Supabase
const fetchProducts = async () => { ... }

// คำนวณราคารวมทั้งหมด
const totalPrice = computed(() => { ... })
```

## UI Text

- ข้อความบน UI ต้องเป็นภาษาไทย
- ชื่อสินค้าอาจมีทั้งไทยและอังกฤษ (ไทยเป็นหลัก)
